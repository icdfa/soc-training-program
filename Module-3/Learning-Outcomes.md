# Module 3: SIEM & Log Management - Learning Outcomes

Upon successful completion of Module 3, students will be able to:

1. **Deploy and configure Splunk** as a SIEM for log management and analysis.
2. **Create correlation searches** in Splunk to detect security incidents.
3. **Integrate threat intelligence** feeds into Splunk for enhanced detection.
4. **Use advanced Splunk searches** to investigate security alerts.
5. **Analyze large-scale SIEM datasets** to identify attack patterns.
