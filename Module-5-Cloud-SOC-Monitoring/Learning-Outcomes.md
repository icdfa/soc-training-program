# Module 5: Cloud SOC Monitoring - Learning Outcomes

Upon successful completion of Module 5, students will be able to:

1. **Understand cloud security concepts** and the challenges of cloud SOC monitoring.
2. **Set up a cloud environment** in AWS for security monitoring.
3. **Analyze cloud security datasets** like CSE-CIC-IDS2018 to identify cloud-based attacks.
4. **Use cloud-native security tools** in AWS, Azure, and GCP for threat detection.
