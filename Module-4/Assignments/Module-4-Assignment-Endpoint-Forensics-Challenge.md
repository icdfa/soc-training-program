# Module 4 Assignment: Endpoint Forensics Challenge

## Objective

In this assignment, you will apply the skills you learned in Module 4 to investigate a compromised endpoint. You will be provided with a memory dump of a Windows machine and a set of questions to answer.

## Scenario

A user has reported that their machine is behaving strangely. They have been experiencing slow performance and have seen suspicious pop-up windows. You have been tasked with analyzing a memory dump of the user's machine to determine if it has been compromised.

## Memory Dump

*   **File:** `memdump.mem` (This will be provided to you)

## Questions

1.  What is the operating system of the machine?
2.  What is the IP address of the machine?
3.  What processes were running on the machine?
4.  Are there any suspicious processes?
5.  What network connections were active?
6.  Are there any suspicious network connections?
7.  What is the name of the malware that infected the machine?
8.  What are the Indicators of Compromise (IOCs) for this malware?

## Deliverables

*   A report answering all of the questions above.
*   Your report should include screenshots and command output to support your answers.
