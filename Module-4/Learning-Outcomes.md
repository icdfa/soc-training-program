# Module 4: Endpoint Security & Malware Analysis - Learning Outcomes

Upon successful completion of Module 4, students will be able to:

1. **Deploy and configure Wazuh** for endpoint security monitoring.
2. **Conduct memory forensics** with Volatility to analyze memory dumps.
3. **Set up FlareVM** for malware analysis.
4. **Perform static and dynamic malware analysis** on real malware samples.
5. **Analyze phishing emails** to identify malicious attachments and links.
