# Module 2 Assignment: Network Forensics Challenge

## Objective

To apply your knowledge of network traffic analysis and forensics to solve a realistic challenge.

## Scenario

You are a SOC analyst and you have been provided with a PCAP file from a network forensics competition. The challenge is to analyze the PCAP file and answer a series of questions about the activity it contains.

## Instructions

1. **Download the Challenge PCAP File:**
   - The PCAP file for this challenge can be downloaded from the `resources` directory of this assignment.

2. **Analyze the PCAP File:**
   - Use Wireshark and any other tools you have learned about in this module to analyze the PCAP file.
   - The questions for the challenge are embedded as comments within the PCAP file. You will need to extract them.

3. **Answer the Questions:**
   - Create a new Markdown document named `Network-Forensics-Challenge-Answers.md`.
   - In this document, provide the answers to the questions from the PCAP file.
   - For each question, provide a brief explanation of how you found the answer, including any Wireshark filters or commands you used.

## Deliverables

- A Markdown document (`Network-Forensics-Challenge-Answers.md`) containing your answers to the challenge questions.
