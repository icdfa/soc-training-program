# Module 2: Network Security & Traffic Analysis - Learning Outcomes

Upon successful completion of Module 2, students will be able to:

1. **Analyze network protocols** and their role in network security monitoring.
2. **Use Wireshark** to capture and analyze network traffic for security incidents.
3. **Deploy and configure Security Onion** for network security monitoring and log management.
4. **Analyze firewall and proxy logs** to identify suspicious activity.
5. **Detect command and control (C2) traffic** in network captures.
6. **Analyze large-scale network datasets** like CIC-IDS2017 to identify attack patterns.
