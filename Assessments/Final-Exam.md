# Final Exam

## Instructions

This is a multiple-choice exam. Please select the best answer for each question.

## Questions

1.  What is the primary function of a SOC?
    a) To develop new security tools
    b) To monitor and respond to security incidents
    c) To perform penetration testing
    d) To write security policies

2.  Which of the following is a common tool used for network traffic analysis?
    a) Splunk
    b) Wireshark
    c) Volatility
    d) Ghidra

3.  What is the purpose of a SIEM?
    a) To collect and analyze log data
    b) To block malicious network traffic
    c) To perform malware analysis
    d) To manage user access controls

4.  Which of the following is a technique used for static malware analysis?
    a) Executing the malware in a sandbox
    b) Analyzing the malware's network traffic
    c) Examining the malware's file properties
    d) Observing the malware's behavior on a live system

5.  What is the first step in the incident response process?
    a) Containment
    b) Eradication
    c) Recovery
    d) Preparation

6.  Which of the following is a common indicator of a phishing email?
    a) The email is from a known contact
    b) The email contains a link to a legitimate website
    c) The email creates a sense of urgency
    d) The email has a personalized greeting

7.  What is the purpose of a threat intelligence platform?
    a) To collect and analyze threat data
    b) To block malicious IP addresses
    c) To perform vulnerability scanning
    d) To manage security policies

8.  Which of the following is a common EDR tool?
    a) Wazuh
    b) Snort
    c) Zeek
    d) Suricata

9.  What is the purpose of a memory dump?
    a) To create a backup of a system's hard drive
    b) To capture the contents of a system's RAM
    c) To log all network traffic to and from a system
    d) To store a system's configuration settings

10. Which of the following is a common framework for threat hunting?
    a) MITRE ATT&CK
    b) NIST Cybersecurity Framework
    c) ISO 27001
    d) CIS Controls
