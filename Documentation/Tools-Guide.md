
# SOC Training Program - Tools Guide

This document provides an overview of the tools used in the SOC Training Program. For detailed instructions on how to use each tool, please refer to the individual lab exercises.

## Virtualization

*   **VirtualBox:** A free and open-source virtualization platform.
*   **VMware Workstation Player:** A free virtualization platform for non-commercial use.

## Network Security Monitoring

*   **Wireshark:** A popular network protocol analyzer.
*   **Security Onion:** A free and open-source platform for network security monitoring and log management.
*   **Zeek:** A powerful network analysis framework.
*   **Suricata:** A high-performance network IDS, IPS, and network security monitoring engine.

## SIEM

*   **Splunk:** A popular SIEM platform.
*   **Wazuh:** A free and open-source security platform that provides threat detection, integrity monitoring, incident response, and compliance.

## Endpoint Security

*   **Volatility:** An open-source memory forensics framework.
*   **FlareVM:** A Windows-based security distribution for malware analysis, incident response, and penetration testing.

## Malware Analysis

*   **Cuckoo Sandbox:** An open-source automated malware analysis system.
*   **Ghidra:** A free and open-source software reverse engineering suite of tools.
*   **IDA Pro:** A popular disassembler and debugger.
