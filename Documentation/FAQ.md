
# SOC Training Program - Frequently Asked Questions (FAQ)

## General Questions

**Q: Who is this program for?**

A: This program is designed for aspiring cybersecurity professionals who want to start a career in a Security Operations Center (SOC). It is also suitable for IT professionals who want to transition into a cybersecurity role.

**Q: What are the prerequisites for this program?**

A: A basic understanding of networking and operating systems is recommended. Familiarity with the command line is also helpful, but not required.

**Q: How long does the program take to complete?**

A: The program is designed to be completed in 6 months (Standard Track), but we also offer a 4-month Fast Track and an 8-month Part-Time Track.

## Technical Questions

**Q: What software do I need for the labs?**

A: You will need VirtualBox or VMware Workstation Player to create and manage virtual machines. All other software used in the labs is free and open-source.

**Q: Can I use a Mac or Linux computer for this program?**

A: Yes, the labs are designed to be run in a virtualized environment, so you can use any operating system that supports VirtualBox or VMware.

**Q: Do I need a powerful computer for this program?**

A: We recommend a computer with at least 16GB of RAM and 100GB of free disk space to run the virtual machines smoothly.
