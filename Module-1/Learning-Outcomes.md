
# Module 1: SOC Fundamentals & Home Lab Setup - Learning Outcomes

Upon successful completion of this module, students will be able to:

*   **Define the role and function of a Security Operations Center (SOC).**
*   **Describe the responsibilities of Tier 1, Tier 2, and Tier 3 SOC analysts.**
*   **Set up a virtual lab environment for hands-on practice.**
*   **Use Linux command-line tools to analyze log files.**
*   **Use Windows PowerShell to analyze event logs.**
