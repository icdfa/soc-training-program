# CTU-13 Dataset

## Overview

The CTU-13 dataset is a labeled dataset with botnet, normal, and background traffic. It was created by the CTU University, Czech Technical University in Prague. The dataset consists of thirteen scenarios of different botnet samples captured in pcap files.

## Dataset Details

**Source:** Czech Technical University in Prague

**Size:** Approximately 32 GB (uncompressed)

**Format:** PCAP files, NetFlow files, and labeled connection logs

**Duration:** 13 different capture scenarios

## Botnet Scenarios

The dataset includes 13 different botnet scenarios:

**Scenario 1:** Neris botnet
**Scenario 2:** Neris botnet (different configuration)
**Scenario 3:** Rbot botnet
**Scenario 4:** Rbot botnet (different configuration)
**Scenario 5:** Virut botnet
**Scenario 6:** Menti botnet
**Scenario 7:** Sogou botnet
**Scenario 8:** Murlo botnet
**Scenario 9:** Neris botnet (long capture)
**Scenario 10:** Rbot botnet (long capture)
**Scenario 11:** Rbot botnet (mixed with normal traffic)
**Scenario 12:** NSIS.ay botnet
**Scenario 13:** Virut botnet (mixed with normal traffic)

## Download Instructions

1. Visit the official CTU-13 dataset page:
   https://www.stratosphereips.org/datasets-ctu13

2. Download the scenarios you need (or all 13)

3. Extract the files to this directory:
   ```
   tar -xzf CTU-13-Dataset.tar.gz -C /path/to/soc-training-program/Datasets/CTU-13/
   ```

## Labels

Each scenario includes labeled connection logs with the following labels:

- **Botnet:** Traffic generated by the botnet
- **Normal:** Legitimate traffic from the infected machine
- **Background:** Traffic from other machines in the network

## Features

The labeled connection logs include:

- Start time
- Duration
- Source IP and port
- Destination IP and port
- Protocol
- State
- Source-to-destination packets
- Source-to-destination bytes
- Destination-to-source packets
- Destination-to-source bytes
- Label (Botnet, Normal, or Background)
