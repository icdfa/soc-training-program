_# Module 7 Quiz: AI/ML in SOC & Capstone Project - Answers

1. **What is the primary purpose of AI and machine learning in a SOC?**
   - AI and machine learning are used in a SOC to automate and augment the work of security analysts. They can be used to identify and prioritize threats, to detect and respond to incidents, and to hunt for threats.

2. **What is the difference between supervised and unsupervised machine learning?**
   - Supervised machine learning is a type of machine learning in which the algorithm is trained on a labeled dataset. This means that the data is already tagged with the correct output. Unsupervised machine learning is a type of machine learning in which the algorithm is not trained on a labeled dataset. Instead, it is allowed to find patterns and relationships in the data on its own.

3. **What is the purpose of a security orchestration, automation, and response (SOAR) platform?**
   - A SOAR platform is a security solution that is used to automate and orchestrate the work of security analysts. It can be used to automate repetitive tasks, to orchestrate the response to incidents, and to provide a single pane of glass for security operations.

4. **What is the purpose of a capstone project in a training program?**
   - A capstone project is a culminating project that is designed to give students the opportunity to apply the knowledge and skills they have learned in a training program to a real-world problem. It is a great way for students to demonstrate their abilities to potential employers.

5. **What are the key components of a professional resume for a SOC analyst?**
   - A professional resume for a SOC analyst should include the following components: a summary of qualifications, a list of technical skills, a list of certifications, a list of work experience, and a list of education.
